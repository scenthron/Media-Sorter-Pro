## üá¨üáß Media Sorter Pro - Advanced Media Organization Tool

**Media Sorter Pro** is a powerful desktop application designed for the visual organization and sorting of large volumes of media content. Created for those who value order, speed, and functionality.

![Main Interface](screenshots/main_interface.png)

### üöÄ Key Features

*   **Non-Destructive Workflow:** Files are moved to a specific "Trash" folder instead of permanent deletion. Undo (`Ctrl+Z`) is always available.
*   **Deep Hierarchy:** Unlimited folder nesting support. Create complex structures (e.g., `Vacation -> 2025 -> Sea`) on the fly.
*   **Freedom of Location:** Sort files from any source (flash drive, network share) to any destination.
*   **Smart Conflict Handling:** Automatic renaming or manual choice when filenames collide.
*   **Real-time Stats:** Monitor file counts and sizes in Inbox, Categories, and Trash.
*   **Affix Panel:** Quick buttons to add prefixes/postfixes to filenames or completely rename them using templates.
*   **Smart Folders (Automation):** Auto-rename files upon moving based on rules (Date, Time, Random Hex, Counter).
*   **Batch Renaming:** Powerful tool to rename all files in a folder using tags.
*   **Media Player:** Built-in viewer with Zoom, Pan, and frame-by-frame video scrubbing.

### üîß Advanced Automation (Tags)

You can use the following tags for **Affix Buttons** and **Smart Folders**:

*   `%date` ‚Äî Date (YYYY-MM-DD)
*   `%time` ‚Äî Time (HH.MM.SS)
*   `%name` ‚Äî Original filename
*   `%seq` ‚Äî Sequential counter (001, 002...)
*   `%rand[4]` ‚Äî Random HEX string (4 chars)
*   `%randNum[3]` ‚Äî Random digits (3 chars)
